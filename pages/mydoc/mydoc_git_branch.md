##Branch
: 코드를 통째로 복사하고 나서 원래 코드와는 상관 없이 독립적으로 개발을 진행하는 것.

```
git checkout branch
  [switch working dir to branch]
```

```
git merge branch
  [merge into current]
```

```
git branch branch
  [branch current]
```

```
git checkout -b new other
  [branch new from other and switch to it]
```
